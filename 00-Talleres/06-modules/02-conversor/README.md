# 2. Conversor de Divisas y Finanzas Personales

Ideal para practicar validaciones (como las que hiciste con el peso y la altura) y lógica de control.

- Archivo `rates.js`: Un objeto o constantes con tipos de cambio (ej. USD a COP, EUR a USD).

- Archivo `converter.js`: Funciones para:

    1. Convertir una cantidad de una moneda a otra.

    2. Formatear el número final a moneda local (usando Intl.NumberFormat o .toFixed()).

    3. Validar que el monto a convertir no sea negativo (lanzar un Error).

- Archivo `main.js`: Simular una lista de gastos en diferentes monedas e imprimir el equivalente total en una sola moneda base.
